# 70 小时提高组交叉强化题单（无模板题版）

## 题单定位

适用对象：

- 已经熟练常见算法模板；
- 不需要再练 STL 基础使用；
- 不想做纯模板题、水题；
- 希望在 70 小时内完成一轮提高组知识点交叉复习；
- 目标是 CSP-S / NOIP T2~T3 水平强化。

本题单特点：

- 不放裸模板题；
- 不放纯 STL 题；
- 不放过于简单的基础题；
- 重点训练建模、转化、DP 状态设计、分类讨论、贪心证明、图树结构分析；
- 难度以「普及+/提高」到「提高+/省选-」为主；
- 少量较有思维价值的综合题。

# 四、70 小时内完成的限时建议

为了保证能在 70 小时内完成，不建议对单题无限死磕。

| 类型 | 单题建议上限 |
|---|---|
| 普及+/提高构造、贪心、分类讨论 | 1.5 小时 |
| 提高+/省选- DP、图论、树上题 | 2.5 小时 |
| 状压 DP / 数位 DP / 树形 DP | 3 小时 |
| P7914、P2831、P3177 这类综合题 | 3.5 小时 |

如果超过上限还没有完整主思路：

1. 先只看题解关键词；
2. 不看代码；
3. 自己补完实现；
4. 做完后记录“没想到的转化”。
